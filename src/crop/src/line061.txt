对拟合直线的参数进行修改

原来是这个
cv::fitLine(centers, line_para, cv::DIST_L2, 0, 1e-2, 1e-2);

新的是
cv::fitLine(centers_change, line_para, cv::DIST_L1, 0, 1e-2, 1e-2);

CV_DIST_L1 =1, /* distance = |x1-x2| + |y1-y2| */
CV_DIST_L2 =2, /* the simple euclidean distance */
